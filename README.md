Belajar JavaScript — Dasar

Repository ini berisi materi dan contoh sederhana untuk mempelajari dasar-dasar JavaScript.

Tujuan
- Menyediakan kumpulan contoh HTML/JS yang mudah diikuti untuk pemula.
- Menjelaskan konsep dasar seperti variabel, tipe data, kontrol alur, dan fungsi.

Struktur proyek (singkat)
- Root: berisi banyak file HTML yang masing-masing memfokuskan topik tertentu.
- `scripts/`: file JavaScript contoh, mis. `hello-world.js`.

Cara menggunakan
- Buka file HTML langsung di browser (klik dua kali atau gunakan `Live Server` di VS Code).
- Untuk melihat contoh JS, buka file di `scripts/` atau lihat tag `<script>` pada file HTML terkait.

Topik yang dibahas (contoh)
- Variabel: `variable.html`, `variable-const.html`
- Kontrol alur: `if-expression.html`, `switch-statement.html`, `for-loop.html`, `while-loop.html`
- Fungsi: `function.html`, `rest-parameter.html`
- Objek & Array: `object.html`, `array.html`
- Operator & Tipe: `operator-aritmatika.html`, `tipe-data-string.html`, `null.html`, `undefined-variable.html`
- Popup & DOM basics: `popup-alert.html`, `popup-confirm.html`, `popup-prompt.html`

Kontribusi
- Perbaikan materi, koreksi penulisan, atau penambahan contoh baru diterima lewat pull request.

Lisensi
- Bebas digunakan untuk belajar dan pengajaran.
